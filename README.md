# FSV Berghausen 1928

Official website for **FSV Berghausen 1928**, built with **Next.js**, **TypeScript**, and **Tailwind CSS**.

---

## Tech Stack

- Next.js (App Router)
- React
- TypeScript
- Tailwind CSS

## Development

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open:

```
http://localhost:3000
```

Build for production:

```bash
npm run build
```

Start production:

```bash
npm run start
```
